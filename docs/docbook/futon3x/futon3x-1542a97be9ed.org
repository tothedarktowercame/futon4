* Context
CRUD operations for the meme store. Provides entity lifecycle management:
=ensure-entity!= (upsert by name+kind), =get-entity=, =list-entities=,
plus artifact and alias management.

Used by the bootstrap script (L1) to populate concepts from hotword extraction.
Each pattern's hotwords become concept entities; co-occurrence creates analogy
arrows between them.

* Context
Federated search across three data sources via core.logic relations:

- =patterno= — searches patterns-index.tsv (hotwords + name)
- =memeo= — SQL LIKE search over meme.db entities (added for L2)
- =transcripto= — searches session transcripts

=search-texto= composes all three with fair-share allocation (limit/3 per source)
to prevent any single source from dominating results. This is the key integration
point where futon3a's meme store becomes queryable through futon3b's relational
interface.

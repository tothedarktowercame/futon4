* Context
~GET /api/alpha/portfolio/state~ returns the current belief state: the mu vector
(15 sensory channels), precision weights, mode, urgency, and inference step
count.

The 15 mu channels measure different aspects of portfolio health:

| Channel                 | What it measures                              |
|-------------------------+-----------------------------------------------|
| mission-complete-ratio  | Fraction of missions marked complete          |
| coverage-pct            | Devmap component coverage by missions         |
| gap-count               | Number of uncovered components (normalized)   |
| blocked-ratio           | Fraction of missions that are blocked         |
| stall-count             | Missions with no recent activity              |
| evidence-velocity       | Rate of new evidence entries                  |
| review-age              | Days since last portfolio review              |
| pattern-reuse           | Fraction of missions using library patterns   |
| dependency-depth        | Max chain length in dependency graph          |
| mana-available          | Resource availability (from nonstarter pool)  |
| spinoff-pressure        | Ratio of spinoff proposals to active missions |
| coverage-trajectory     | Is coverage improving or declining?           |
| unplanned-work-ratio    | Fraction of work not tied to a mission        |
| effort-prediction-error | Gap between bid and actual effort             |
| bid-completion-rate     | Fraction of weekly bids completed             |

Each channel has a precision weight (Pi-o) that determines how much the channel
contributes to free energy computation. High-precision channels dominate the
inference; low-precision channels are effectively ignored.

All mu channels are at 0.5 (prior) because this is a fresh instance — no
portfolio steps have run yet to update beliefs from observations. The precision
weights show the mode-conditioned attention: in BUILD mode, ~mission-complete-ratio~,
~coverage-pct~, ~blocked-ratio~, and ~review-age~ have the highest precision
(1.0), while ~pattern-reuse~ and ~spinoff-pressure~ are attenuated (0.3, 0.4).

* Context
~GET /api/alpha/mc/trace~ traces every current tension through a 6-gate chain.
Each gate checks one link in the path from structural claim to source code. The
trace answers: "how far can we get from this tension before the chain breaks?"

The six gates are:

| Gate       | Checks                        | Pass means                      |
|------------+-------------------------------+---------------------------------|
| devmap     | Does the devmap exist?        | Structural skeleton is defined  |
| component  | Does the component exist?     | The gap has a name and position |
| coverage   | Is there a covering mission?  | Someone is addressing it        |
| evidence   | Does the parent mission exist | Work has been attempted         |
| reflection | Is there loaded Clojure code? | Implementation exists in JVM    |
| source     | Can we reach file + line?     | Claim bottoms out at code       |

A complete trace means a tension can be navigated from structural specification
all the way to a line of source code. A gap at ~coverage~ means nobody is
addressing the component (間: the honest interval). A block at ~reflection~
means no code exists yet (関: the gate that hasn't been traversed).

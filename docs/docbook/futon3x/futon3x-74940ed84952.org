* Context
Pattern matching predicates used by gate pipeline stages. Provides the structural
checks that determine whether a work product satisfies gate constraints — e.g.,
whether a PSR references a valid pattern from the library, whether execution
evidence has the required shape.

Works with =futon3.gate.pipeline= to enforce the gate sequence. Patterns are
resolved via futon3a's pattern index (patterns-index.tsv) through the federated
search layer.

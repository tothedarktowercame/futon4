* Context
`futon3.hx.validate` wraps the logic engine and logs validation outcomes into
`futon3.logic-audit`. `structural-step` is the central helper; it runs
`hx.logic/check-step`, records the result, and returns a minimal validation map
for callers.

The module also builds higher-level validation payloads for link suggestion
flows. `self-validation` packages an agent's decision plus structural checks,
while `peer-review` attaches a second sign-off for cross-agent review.

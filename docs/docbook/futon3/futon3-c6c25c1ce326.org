* Context
`f2.semantics` is a small analytics layer over transport history. It surfaces
shared outputs across clients (`suggest-links`), instantiates naive pattern
templates from recent client entries (`instantiate-pattern`), and provides a
basic churn check (`check-consistency`) for noisy clients.

The functions are intentionally lightweight and operate purely on the in-memory
history vector so the UI can compute insights without reaching into storage.

* Context
`futon3.tatami` is the interactive and programmatic logger for Tatami sessions.
It maintains a current session atom plus a default selection (prototypes and
intent), and provides `start-session`, `log-event`, and `close-session` as the
programmatic API. The interactive `start`/`log`/`close` functions wrap the same
logic with prompts.

Events are normalized through `futon3.tatami-schema` and written via
`futon3.tatami-store`. The module also derives fruit/paramita labels based on
activity and felt-state, and `convolve` runs simple heuristics to detect
candidate evidence for higher-level proofs. `status` aggregates the last 24
hours and can include stack telemetry and current selection.

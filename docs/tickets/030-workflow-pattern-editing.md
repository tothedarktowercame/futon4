## What needs documenting
Step-by-step workflow:
1. Use browser to navigate to a pattern
2. Press RET to open in Org buffer
3. Edit the pattern text
4. Use `g` to refresh from source
5. Use `C-c C-s` to save back
6. Sync happens if enabled
